# Sistema di Gestione Spese Personali

## Descrizione
Applicazione da console sviluppata in Python per la gestione delle spese personali e del budget mensile.

## Tecnologie utilizzate
- Python
- SQLite

## Funzionalità
- Gestione categorie
- Inserimento spese
- Eliminazione spese
- Definizione budget mensile
- Eliminazione categorie
- Report:
  - Totale spese per categoria
  - Spese vs budget
  - Elenco completo spese

## Avvio del programma

Eseguire il comando:

python main.py
